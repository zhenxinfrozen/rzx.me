### 1. 路由命名规范
```

sketchbook → sketchbook-new  ✅

video-gallery → videos-new  ✅（注意：不是video-gallery-new）
```

### 2. 控制器路径规范
所有AJAX请求的控制器路径：
```javascript
// ✅ 正确格式（用短横线，无空格）

const controllerUrl = '/admin/ajax?controller=comics';

// ❌ 错误格式

```

## ⚡ 常见错误排查

### 问题1：页面跳转到首页
**原因**：
- 路由未配置或配置错误
- 视图文件不存在
- 控制器加载失败

**解决**：
1. 检查 `AdminIndexController.php` 中的 `$pages` 数组
2. 检查视图文件路径是否正确
3. 检查控制器文件是否存在

### 问题2：AJAX请求失败（404或500）
**原因**：
- JavaScript中的 `controller=` 参数有空格或拼写错误
- 控制器文件路径不对

**解决**：
1. 全局搜索 `controller=`，确保无空格
2. 确认控制器文件在 `app/Admin/Controllers/` 目录下
3. 检查控制器文件名是否与路由一致

### 问题3：数据加载失败
**原因**：
- bootstrap.php 未加载
- 数据文件路径错误
- 控制器未正确执行

**解决**：
1. 确保新页面在 `AdminIndexController.php` 的 bootstrap 加载列表中
2. 检查数据文件是否存在于 `app/storage/data/`
3. 查看浏览器控制台和PHP错误日志
