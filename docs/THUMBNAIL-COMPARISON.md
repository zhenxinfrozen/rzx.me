# 视频缩略图方案对比 - FFmpeg vs GD库

## 🎬 真实截图 vs 程序生成图

### 你看到的黑色/灰色图片确实是"程序生成"的，不是视频截图！

## 📊 视觉效果对比

### GD库方案（当前使用）

```
┌─────────────────────────┐
│    #282828 深灰背景     │
│                         │
│         ▶               │  ← 白色三角形（程序画的）
```

**特点**：

```
   从视频中提取的真实帧
```

**特点**：
- ✅ 真实视频画面
- ✅ 原始色彩
- ✅ 实际内容预览

## 💻 代码层面对比

### GD库生成代码（程序画图）

```php
// 创建一张空白图片
$image = imagecreatetruecolor(320, 180);

// 填充深灰色背景
$bgColor = imagecolorallocate($image, 40, 40, 40);
imagefill($image, 0, 0, $bgColor);

// 画一个白色三角形
$iconColor = imagecolorallocate($image, 255, 255, 255);
imagefilledpolygon($image, $points, 3, $iconColor);

// 写上文件名
imagestring($image, 3, $x, $y, "Video Name", $textColor);

// 保存为JPG
imagejpeg($image, 'output.jpg');
```

**这就是为什么你看到的都是灰色图片！**

### FFmpeg提取代码（真实截图）

```bash
# 从视频的第2.5秒提取一帧
ffmpeg -ss 2.5 -i video.mp4 -vframes 1 -vf "scale=320:-1" -q:v 2 output.jpg
```

**这才是真正的视频截图！**

## 💡 解决方案

### 方案A: 本地也安装FFmpeg（可选）

如果想在本地看到真实效果：

**Windows安装步骤**：
1. 下载: https://www.gyan.dev/ffmpeg/builds/
2. 解压到: `C:\ffmpeg`
3. 添加到PATH: `C:\ffmpeg\bin`
4. 重启PowerShell
5. 运行: `php regenerate-thumbnails.php`

**效果**：
```
之前: [灰色图片]  [灰色图片]  [灰色图片]
之后: [真实画面]  [真实画面]  [真实画面]
```

### 方案B: 等VPS部署时自动切换（推荐）

**流程**：
```
本地开发:
  - 继续用GD库灰色图
  - 功能开发和测试

VPS部署:
  - sudo apt-get install ffmpeg
  - php regenerate-thumbnails.php
  - 自动变成真实截图
```
