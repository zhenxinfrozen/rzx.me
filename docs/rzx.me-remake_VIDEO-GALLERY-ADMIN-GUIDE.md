# Video Gallery 后台管理系统 - 使用说明

## 📋 功能概览

Video Gallery 后台管理系统已成功集成到 RZX.ME 后台管理系统中，提供以下功能：

### ✅ 已实现功能

1. **分组管理**
   - 创建新视频分组
   - 编辑分组名称和描述
   - 删除空分组
   - 拖拽排序分组

2. **缩略图管理**
   - 自动使用分组内第一个视频的缩略图作为分组封面
   - 支持重新生成分组缩略图
   - 支持批量生成所有缩略图

3. **视频扫描**
   - 扫描所有视频目录并更新数据库
   - 自动检测新增视频文件

4. **前台预览**
   - 直接链接到前台视频页面
   - 实时查看修改效果

## 🎯 使用指南

### 1. 创建新视频分组

1. 点击左侧 **"分组顺序"** 卡片右上角的 **绿色 + 按钮**
2. 填写分组信息：
   - **新分组名称**: 英文文件夹名（如 `my-animations`）
   - **显示名称**: 显示给用户的中文名称（如 `我的动画`）
   - **描述信息**: 可选的分组描述
   - **排序位置**: 选择 "第一个" 或 "最后一个"
3. 点击 **"创建分组"** 按钮
4. 系统会自动创建对应的文件夹：`public/assets/videos/video-gallery/my-animations/`

### 2. 编辑分组信息

1. 点击左侧分组列表中的任意分组
2. 右侧会显示编辑面板
3. 修改 **显示名称** 或 **描述信息**
4. 点击 **"更新"** 按钮保存

### 3. 调整分组顺序

1. 在左侧分组列表中，按住分组左侧的 **⋮⋮ 拖拽手柄**
2. 拖动分组到目标位置
3. 松开鼠标，系统自动保存新顺序
4. 前台页面会立即反映新的顺序

### 4. 重新生成缩略图

#### 方法一：重新生成单个分组
1. 选择要处理的分组
2. 点击 **"重新生成缩略图"** 按钮
3. 等待生成完成

#### 方法二：批量生成所有缩略图
1. 点击右侧工具栏的 **"重新生成所有缩略图"** 按钮
2. 复制弹出的命令
3. 在服务器终端执行：
   ```bash
   php app/Console/GenerateThumbnails.php -f
   ```

### 5. 扫描新增视频

当你直接在文件系统中添加了新视频文件后：

1. 点击右侧工具栏的 **"扫描所有视频"** 按钮
2. 系统会自动扫描所有视频目录
3. 发现新视频后会自动更新数据库
4. 页面会自动刷新显示新数据

### 6. 删除分组

**注意：只能删除空分组！**

1. 选择要删除的分组
2. 点击 **"删除分组"** 按钮
3. 确认删除操作
4. 如果分组内有视频文件，系统会提示先手动删除视频

## 🔧 技术细节

### API 接口

控制器提供以下 AJAX 接口：

| 接口 | 方法 | 说明 |
|------|------|------|
| `?ajax=videos` | GET | 获取分组的视频列表 |
| `?ajax=create_group` | POST | 创建新分组 |
| `?ajax=save_group` | POST | 保存分组更新 |
| `?ajax=delete_group` | POST | 删除分组 |
| `?ajax=regenerate_thumbnails` | GET | 重新生成分组缩略图 |
| `?ajax=scan_directory` | GET | 扫描视频目录 |

### 数据流

```
1. 用户操作 (前端 JavaScript)
   ↓
2. AJAX 请求 (body-video-gallery.php)
   ↓
3. 控制器处理 (video-gallery.php)
   ↓
4. 调用模型 (app/Models/video_data.php)
   ↓
5. 读写配置 (app/Config/video_gallery_sort.php)
   ↓
6. 返回 JSON 响应
   ↓
7. 前端更新界面
```

### 配置文件结构

`app/Config/video_gallery_sort.php`:
```php
return [
    'sort_method' => 'custom_order',
    'custom_order' => ['storyboard', 'animation-clips', ...],
    'display_names' => [
        'storyboard' => '分镜',
        'animation-clips' => '动画片段',
        ...
    ],
    'descriptions' => [
        'storyboard' => '故事板和分镜草图',
        ...
    ],
];
```

## 🐛 常见问题

### Q1: 为什么分组缩略图不显示？

**A:** 分组缩略图默认使用该分组内第一个视频的缩略图。如果没有显示，可能是：
1. 分组内还没有视频文件
2. 视频缩略图还未生成
3. 解决方法：点击 **"重新生成缩略图"** 按钮

### Q2: 为什么删除分组失败？

**A:** 系统只允许删除空分组。如果分组内有视频文件，需要：
1. 手动删除文件夹内的所有视频文件
2. 或者将视频移动到其他分组
3. 然后再删除分组

### Q3: 为什么拖拽排序不生效？

**A:** 检查以下几点：
1. 确保 JavaScript 已加载（F12 控制台查看错误）
2. 确保按住了 **⋮⋮ 拖拽手柄** 而不是其他区域
3. 刷新页面重试

### Q4: 如何批量上传视频？

**A:** 目前不支持在后台直接上传视频。建议：
1. 使用 FTP 或文件管理器上传视频到对应分组目录
2. 上传后点击 **"扫描所有视频"** 按钮
3. 系统会自动检测新视频并生成缩略图

### Q5: 前台页面没有显示新分组？

**A:** 检查以下步骤：
1. 确认分组配置已保存（检查 `app/Config/video_gallery_sort.php`）
2. 确认分组内至少有一个视频文件
3. 点击 **"扫描所有视频"** 更新数据库
4. 刷新前台页面（Ctrl+F5 强制刷新）

## 🔄 与其他管理页面的一致性

Video Gallery 管理页面遵循与 Single-Works、Sketchbook 相同的设计模式：

### 布局结构
- **左侧**: 分组列表（支持拖拽排序）

- **绿色卡片头**: 分组列表

- **青色卡片头**: 预览区域
