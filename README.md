rzx.me
======
项目版本：v3.0.0
历史时期：（2009-2012）大学期间

简介：
纯手敲代码，静态页面，保留手绘风格与创意小细节。使用 HTML/CSS/JavaScript（部分历史页面曾使用 Flash，现以 JS/静态替代），用于展示漫画、原画与练习作品。

当初域名：www.ruizhenxin.com
