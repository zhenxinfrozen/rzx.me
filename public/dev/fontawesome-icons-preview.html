🎨 Font Awesome 免费图标库预览

Font Awesome 6 免费版（Solid/Regular/Brands）图标一览，点击可复制 HTML 代码

📖 使用说明

在 HTML 中引入 Font Awesome CDN，然后用 <i class="fa-solid fa-xxx"></i> 或 fa-regular、fa-brands 使用：

<!-- 引入 Font Awesome CDN --> <link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.4.2/css/all.min.css"> <!-- 使用图标 --> <i class="fa-solid fa-user"></i> <i class="fa-regular fa-image"></i> <i class="fa-brands fa-github"></i>
加载中... 个图标可用
代码已复制到剪贴板！
